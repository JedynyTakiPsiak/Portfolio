## Praca inżynierska
[Przejdź do projektu](./Pinned/Praca-Inzynierska)

Praktyczna część mojej pracy inżynierskiej dotyczyła stworzenia systemu do automatycznego pobierania, analizy statystycznej i wizualizacji wyników gry Lotto.

Najważniejsze elementy:

- pobieranie i lokalny zapis danych historycznych,
- analiza częstotliwości liczb i kombinacji,
- testy statystyczne, m.in. chi-kwadrat i Kołmogorow-Smirnow,
- generowanie końcowego raportu HTML.

Cel projektu:
- połączenie programowania, analizy danych i wizualizacji wyników w jednym spójnym systemie.

## Engineering thesis

[Go to project](./Pinned/Praca-Inzynierska)

The practical part of my engineering thesis focused on building a system for automatic data collection, statistical analysis, and visualization of Lotto results.

Main features:
- C/C++ with external API integration,
- downloading and storing historical data locally,
- frequency analysis of numbers and combinations,
- statistical tests such as chi-square and Kolmogorov-Smirnov,
- generation of a final HTML report.

Project goal:
- combining programming, data analysis, and result visualization into one consistent system.

## Programs in C and C++

[Go to folder](./Studies/Programs_in_C_Cplusplus)

A set of smaller programs showing my learning process in algorithms, data structures, and inter-process communication.

#### A. FIFO communication
- communication between two independent consoles,
- use of FIFO and files,
- practice of basic IPC concepts.

#### B. Finding and removing the smallest element from a heap
- implementation of removing the smallest heap element,
- use of `heapify` and `extractMin`,
- comparison of measured results with theory,
- analysis of the practical efficiency of the data structure.

#### C. Solving a mathematical expression
- generating or entering custom expressions,
- evaluation using Reverse Polish Notation,
- use of stacks,
- practice of basic parsing and algorithmic concepts.
